# FreeRTOS 实时四旋翼飞控

## 1. 项目简介
本项目是一款基于 **STM32F103** 芯片与 **FreeRTOS** 实时操作系统开发的四旋翼无人机飞控系统。系统采用模块化设计，通过串级 PID 算法与姿态解算，实现高频姿态自稳、定高悬停功能。

## 2. 核心技术栈
* **内核**: FreeRTOS (抢占式调度)
* **芯片**: STM32F103 (Cortex-M3)
* **传感器**: BMI088 (IMU)、定点光流模块
* **算法**: 互补滤波/卡尔曼滤波姿态解算、四元数转欧拉角、串级 PID 控制
* **通讯**: NRF24L01 (2.4G 遥控)

## 3. 软件架构设计 (FreeRTOS 应用)
系统充分利用 FreeRTOS 内核特性，保证控制回路的确定性与实时性：

| 任务名称 | 优先级 | 周期 | 核心功能 | 核心 FreeRTOS 知识 |
| :--- | :--- | :--- | :--- | :--- |
| **ControlTask** | Realtime | 5ms | 姿态解算、串级 PID、电机控制 | 任务优先级划分、时序管理 |
| **CommTask** | High | 10ms | NRF24L01 数据接收与解析 | 任务间逻辑解耦 |
| **OtherTask** | Low | 100ms | 电池电压监控、LED 状态指示 | 低功耗延时处理 |

### 技术亮点：
* **数据流解耦**: 采用 `Stream Buffer` 配合串口中断，解决高频串口数据（光流）接收时产生的丢包与解析延迟问题。
* **精细化初始化**: 遵守 FreeRTOS 规范，在 `osKernelStart` 前完成内核对象（如流缓冲区）创建，确保中断安全性。

## 4. 硬件架构
采用**分布式设计理念**：
* **感知层**: BMI088 六轴传感器提供高精度原始惯性数据。
* **处理层**: STM32 执行复杂的浮点运算与控制指令。
* **执行层**: 4 路 PWM 驱动高速无刷电机。

## 5. 开发工具
* **IDE**: VS Code + CMake / Keil MDK
* **调试**: dap-link / Ozone / STM32CubeMX
